Sistema para gerencimento/cadastro de funcionários, onde o operador do sistema tem total liberdade para realizar ações como: Verificação, Inclusão, Atualização e Deleção de Departamentos, Cargos e/ou Funcionários. Ferramenta de fácil manuseio e compreensão, otimizando assim o tempo necessário demandado para a realização das tarefas referentes ao gerenciamento de funcionários.
Sessão designada para o gerenciamento de Departamentos. Na aba "Listar" são possíveis as ações de visualizar departamentos já cadastrados, editá-los ou removê-los. Já na aba "Cadastrar", podem ser incluídos novos departamentos.
Sessão designada para o gerenciamento de Cargos. Na aba "Listar" são possíveis as ações de visualizar cargos já cadastrados e seus respectivos departamentos relacionados, além de editá-los ou removê-los. Já na aba "Cadastrar", podem ser incluídos novos cargos e seus respectivos departamentos relacionados.
Sessão designada para o gerenciamento de Funcionários. Na aba "Listar" são possíveis as ações de visualizar funcionários já cadastrados, editá-los ou removê-los. Já na aba "Cadastrar", podem ser incluídos novos funcionários.
Sistema desenvolvido em Java, utilizando as facilidades do Framework Spring Boot e Spring MVC. Para Mapeamento Objeto relacional foi utilizado JPA(Hibernate), como gerenciador de dependências o Maven, e MySQL como banco de dados relacional.
Template engine: Thymeleaf. Template desenvolvido em HTML, CSS e JS.
